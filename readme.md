[WIP] 一个简易的 docker 判题机实现

- solution 是对用户提交操作的抽象，提供读写用户提交以及题目信息的操作
- actuator 编译并执行用户代码，获取用户的输出，对比用户的输出是否与标准输出一致，返回对比结果
- app 中为控制功能，初始化数据库连接，多久轮询一次是否存在提交，对比结果如何回写等都在这里面实现
